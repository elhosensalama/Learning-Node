@ More Operations on Packages using .

- when we want to check what are the packages which need to be updated.
-- we use .
[elhosen@elhosen-fedora ~]$ npm outdated
Package Current Wanted Latest Location
  slugify     1.0.0    1.0.2    1.3.4   learning-node
[elhosen@elhosen-fedora ~]$
Note : package like this slugify@1.2.5 this three numbers have meaning.
: ` 1 ` is the version and when it chnages, old versions code will make Bugs.
: ` 2 ` is the number of add new features and when it chnages, old code will work Fine.
: ` 5 ` is the number of fixed bugs and when it chnages, old code will work Fine Too.
- when we want to updated an outdated package.
-- we use
[elhosen@elhosen-fedora ~]$ npm update slugify
npm Pla Pla Pla...
npm WARN Pla Pla Pla...
