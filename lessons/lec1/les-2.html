Q1: How to install NodeJS ?

  1. 1 - Go To nodejs.org site.
  2. 2 - The Site will automaticly Select Your OS.
  3. 3 - Download the LTS (Long Term Support) version.
  4. 4 - The Recommended Version for Production is the Even Version.
  5. 5 - when Download is finished, Install Node.
  6. 6 - to test if nodejs is Installed use this command node --version .
    7. [elhosen@elhosen-fedora ~]$ node --version
    v16.14.0
